

* * *

🧠 题目理解
-------

给定一个长度为 `n` 的数组 `a`，我们要求对每个 `k` ∈ [0, n] 输出一个值 `cnt[k]`，表示满足某些条件的 `i` 有多少个（再加1输出）。

具体处理逻辑：

1. 统计每个值出现的频次：`freq[x] = 出现次数`；

2. 找到 **mex** 值 `p`：即最小的 `i` 使得 `freq[i] == 0`；

3. 对于所有 `i ∈ [0, p-1]`，如果对某个 `k` 满足 `k ∈ [freq[i], n-i-1]`，则 `cnt[k]++`；

4. 最终输出 `cnt[k] + 1`（因为初始值为1）。

* * *

🐌 暴力做法（会超时）
------------

    for (int k = 0; k <= n; k++) {
        int cnt = 1;
        for (int i = 0; i < p; i++) {
            if (k >= freq[i] && k <= n - i - 1) cnt++;
        }
        cout << cnt << " ";
    }

* 时间复杂度：**O(n²)**，最坏时 `p ≈ n`，嵌套双循环；

* 对于数据规模大时（如 `n = 1e5`），会 **TLE（超时）**。

* * *

⚡ 优化思路：差分数组 + 前缀和
-----------------

为了避免 O(n²)，我们引入差分数组，优化统计 `cnt[k]++` 的过程：

### ✅ 差分数组思想

要对区间 [L, R] 内的所有 `k` 做 `cnt[k]++`，我们只在差分数组中：
    diff[L] += 1;
    diff[R+1] -= 1;

**最后对差分数组做前缀和，就能还原出 cnt[k] 的值。**

### ✨ 差分优化流程

1. **统计 freq 数组：**
      vector<int> freq(n+1);
      for (int x : a)
   
          if (x <= n) freq[x]++;

2. **找到 mex 值：**
      int p = 0;
      while (p <= n && freq[p] > 0) p++;

3. **差分操作（记录每个区间）：**
      vector<int> diff(n + 2, 0); // 多开一位
      for (int i = 0; i < p; i++) {
   
          int L = freq[i], R = n - i - 1;
          if (L <= R) {
              diff[L]++;
              diff[R + 1]--;
          }
   
      }

4. **前缀和还原 + 输出答案：**
      int cur = 0;
      for (int k = 0; k <= n; k++) {
   
          cur += diff[k];
          cout << (1 + cur) << " ";
   
      }
   
   

* * *

⏱️ 时间复杂度分析
----------

* 统计频率 freq：O(n)

* 构造差分：O(p) ≤ O(n)

* 计算前缀和：O(n)

* 输出：O(n)

### ✅ 总体复杂度：**O(n)** —— 可通过所有测试点！

* * *

📝 总结一句话
--------

> 通过将「区间加法」转化为差分数组修改 + 前缀和，我们将暴力双循环 O(n²) 降为 O(n)，成功避免超时，是竞赛中常用的技巧！
